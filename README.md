# Hand-Gesture-Translator
An IoT project integrated with AI Model to translate American and arabic sign language and send the letters to ESP8266 by websocket protocol which is a real time communication protocol. 
The ESP then send the letter to the aurduino to be displayed on the lcd. The LCD displays if a letter is deleted or if the word/sentence cleared. 
In addition ,the ESP sends the letter to an MIT application through websocket protocol. The application displayes the letters and there is a button which enables the speech of the word, it has another functionality which is writing a word in a text box and it begins to diplay pictures of each sign of the word or sentence therefore it has two way communication
